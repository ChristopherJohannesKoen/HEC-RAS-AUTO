# Prompt Live Run Example

This folder contains one curated example output package from `run_id=prompt_live_run`.

Contents:
- `baseline/longitudinal_profile.png`: baseline profile plot.
- `baseline/metrics.csv`: baseline hydraulic metrics.
- `baseline/energy_floodline.geojson`: baseline floodline geometry.
- `baseline/floodlines.dxf`: CAD floodline export.
- `scenario2/scenario2_tier_comparison.csv`: Scenario 2 triad comparison table.
- `scenario2/scenario2_tier_overlay_profile.png`: Scenario 2 triad profile overlay.
- `reports/prompt_live_run_report_draft.md`: baseline report draft.
- `reports/prompt_live_run_scenario_2_triad_report_draft.md`: scenario triad report draft.
- `qa/*.md`: geometry/hydraulic QA notes used in this example.

The full pipeline to regenerate these outputs is documented in the repository root `README.md`.
